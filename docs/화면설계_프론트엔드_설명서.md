## 1. 프로젝트 개요

### 시스템 구성도

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│  Flutter App    │     │  Admin Dashboard │     │  User Service   │
│  (Android/iOS)  │     │  (React Vite)    │     │  (Spring Boot)  │
│   Port: N/A     │     │   Port: 3001     │     │   Port: 8080    │
└────────┬────────┘     └────────┬─────────┘     └────────┬────────┘
         │                       │                        │
         └───────────────────────┼────────────────────────┘
                                 ▼
                    ┌─────────────────────────┐
                    │      ML 서버 (FastAPI)   │
                    │  run_server.py : 8000   │
                    │                         │
                    │  - 시세 예측 (ML 모델)   │
                    │  - Groq AI 분석          │
                    │  - 추천 시스템           │
                    │  - 차량 이미지 서빙      │
                    │  - Admin API            │
                    └─────────────────────────┘
```

### 사용 기술 스택

| 구분 | 기술 |
|------|------|
| **모바일 앱** | Flutter 3.x, Dart, Provider 상태관리 |
| **관리자 대시보드** | React 18, Vite, Lucide Icons |
| **ML 서버** | FastAPI, Python, SQLite |
| **인증 서버** | Spring Boot 3.x, JWT, OAuth2 |
| **AI 엔진** | Groq API (Llama 3.3 70B) |

```
│  └─────┘ └─────┘ └─────┘              │

```

#### 주요 기능

| 기능 | 설명 |
|------|------|
| **이메일/비밀번호 로그인** | Spring Boot User Service 연동 (JWT 발급) |
| **소셜 로그인** | 네이버(N), 카카오(K), 구글(G) OAuth2 지원 |
| **회원가입** | SignupPage로 이동하여 신규 가입 |
| **바로 시세 조회** | 로그인 없이 "내 차 찾기" 탭으로 이동 |
| **최근 조회 차량** | RecentViewsProvider에서 관리, 클릭 시 상세 분석 모달 |
| **인기 모델 추천** | PopularCarsProvider에서 API 호출, 실매물 목록 모달 |

#### 상태 관리 (Provider)

```dart
// 최근 조회 차량
Consumer<RecentViewsProvider>

// 인기 모델 추천
Consumer<PopularCarsProvider>
```
