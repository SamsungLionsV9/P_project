# B2B Market Intelligence 서비스

## 📊 개요

B2B Market Intelligence는 **기업 고객(딜러사, 금융사, 렌터카)**을 위한 데이터 인사이트 서비스입니다.

### 타겟 고객
| 고객 유형 | 활용 목적 |
|:---|:---|
| **중고차 딜러사** | 매집/매각 타이밍 최적화, 재고 관리 |
| **캐피탈/리스사** | 잔존가치 예측, 대출 심사 로직 |
| **렌터카 업체** | 차량 교체 시점 결정, 포트폴리오 관리 |
| **보험사** | 차량 가치 평가, 손해 산정 |

## 🔢 데이터 구성

### 1. 시장 기회 지수 (Market Opportunity Index)

**계산 공식:**
```
MOI = 기본점수(50) + 유가영향(±15) + 환율영향(±15) + 금리영향(±20) + 계절성(±10)
```

**데이터 소스:**
| 지표 | 소스 | 가중치 |
|:---|:---|:---|
| 유가 (WTI) | Yahoo Finance API | 30% |
| 환율 (USD/KRW) | Yahoo Finance API | 30% |
| 기준금리 | 한국은행 Open API | 40% |

**점수 해석:**
- **70~100점**: 적극 매수 (저평가 매물 많음)
- **50~69점**: 관망 (시장 평균)
- **0~49점**: 매수 주의 (고평가 구간)

---

### 2. 매집 추천 (Buying Signals)

**계산 로직:**
```python
signal_score = (
    가격변동률_점수 * 0.35 +    # 최근 30일 가격 하락폭
    수요트렌드_점수 * 0.25 +    # 검색량/조회수 추이
    재고회전율_점수 * 0.20 +    # 시장 재고 대비 거래량
    경제지표_점수 * 0.20        # MOI 기반
)

추천_강도 = {
    '강력 추천': signal_score >= 80,
    '추천': signal_score >= 65,
    '관심': signal_score >= 50
}
```

**출력 데이터:**
```json
{
  "model": "쏘렌토 MQ4",
  "signal_strength": "강력 추천",
  "expected_roi": 12.5,
  "risk_level": "LOW",
  "reasoning": "SUV 수요 증가 + 유가 하락세",
  "target_price": 3200,
  "holding_period": "3개월"
}
```

### 3. 매각 경고 (Sell Signals)

**위험 감지 로직:**
```python
risk_factors = {
    '신차 출시 임박': new_model_release_days < 90,
    '감가 가속화': depreciation_rate > threshold,
    '수요 감소': demand_trend == 'declining',
    '계절 비수기': is_off_season,
    '경제지표 악화': moi < 40
}

urgency = sum(risk_factors.values())
```

**출력 데이터:**
```json
{
  "model": "제네시스 G80",
  "urgency": "HIGH",
  "expected_loss_if_delayed": -8.5,
  "risk_factors": ["신차 출시 90일 내", "럭셔리 수요 감소"],
  "recommended_action": "60일 내 매각 권장",
  "optimal_price": 4200
}
```

### 4. 포트폴리오 ROI 분석

**수익률 계산:**
```
예상 ROI = (예상 매각가 - 매입가 - 보유비용) / 매입가 × 100

보유비용 = (
    감가상각비 +      # 일평균 × 보유일수
    자본비용 +        # 매입가 × 금리 × 보유기간
    기회비용          # 대안 투자 수익률
)
```

**세그먼트별 기대 ROI:**
| 세그먼트 | 평균 ROI | 위험도 | 추천 보유기간 |
|:---|:---|:---|:---|
| EV (전기차) | 8~15% | HIGH | 2-4개월 |
| SUV | 6~12% | MEDIUM | 3-6개월 |
| 세단 | 4~8% | LOW | 4-8개월 |
| 럭셔리 | 5~10% | HIGH | 2-4개월 |

---

### 5. 예측 정확도

**검증 방법:**
```
정확도 = 1 - |실제가격 - 예측가격| / 실제가격 × 100

7일 정확도: 최근 7일간 예측 vs 실제 거래가
30일 정확도: 최근 30일간 예측 vs 실제 거래가
```

**현재 성능 (시뮬레이션 기준):**
| 예측 기간 | 정확도 | 샘플 수 |
|:---|:---|:---|
| 7일 | 94.2% | 1,247건 |
| 30일 | 91.8% | 892건 |
| 90일 | 87.5% | 423건 |

### 6. 민감도 분석

**경제지표별 가격 영향:**
```python
SENSITIVITY_MATRIX = {
    'large_sedan': {'금리': -15%, '유가': -5%, '환율': -3%},
    'mid_suv': {'금리': -8%, '유가': -8%, '환율': -3%},
    'compact_suv': {'금리': -6%, '유가': -10%, '환율': -2%},
    'ev': {'금리': -10%, '유가': +15%, '환율': -5%},  # EV는 유가↑ = 수요↑
    'luxury': {'금리': -20%, '유가': -3%, '환율': -8%},
}
```

**해석 예시:**
- 금리 1%p 상승 시 → 대형 세단 가격 15% 하락 예상
- 유가 10% 상승 시 → EV 가격 15% 상승 예상 (전환 수요)
