# 중고차 가격 예측 모델 전략

## 📊 현재 상황 분석

### 문제점
1. **국산차 단일 모델**로는 일반 국산차와 제네시스를 동시에 커버하기 어려움
2. 가격대가 100만원~2억원으로 너무 넓음
3. 제네시스는 수입차와 유사한 패턴을 가짐

### 검증 결과
- 일반 국산차 (100만~5000만): R² 0.93, MAE 185만원
- 전체 국산차 (100만~2억): R² 0.60, MAE 377만원 ⬇️
- 수입차 (전체 범위): R² 0.99, MAE 63만원

---

## 🎯 제안: 3-Tier 모델 시스템

### Model 1: 일반 국산차 모델
- **대상**: 현대, 기아, 쉐보레, 르노, KG모빌리티 등
- **가격대**: 100만~8,000만원
- **데이터**: 약 110,000건
- **예상 성능**: R² 0.93+, MAE 180만원
- **특화 피처**: 주행거리, 연식, 빠른 감가상각

### Model 2: 프리미엄 국산차 모델 (제네시스)
- **대상**: 제네시스 브랜드
- **가격대**: 1,000만~2억원
- **데이터**: 약 10,000건
- **예상 성능**: R² 0.90+, MAE 300만원
- **특화 피처**: 브랜드 프리미엄, 모델명, 희소성

### Model 3: 수입차 모델
- **대상**: 모든 수입 브랜드
- **가격대**: 100만~10억원
- **데이터**: 49,055건
- **현재 성능**: R² 0.99, MAE 63만원 ✅
- **특화 피처**: 럭셔리 브랜드, 브랜드 가치, 모델 희소성
