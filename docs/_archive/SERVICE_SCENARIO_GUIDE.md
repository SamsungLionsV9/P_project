## 1. 서비스 개요

### 1.1 서비스 목적

Car-Sentix는 중고차 구매자를 위한 **데이터 기반 의사결정 지원 서비스**입니다.

| 핵심 가치 | 설명 |
|-----------|------|
| **가격 적정성 판단** | ML 모델 기반 시세 예측으로 매물 가격이 적정한지 판단 |
| **구매 시점 최적화** | 거시경제 지표 분석으로 지금이 구매 적기인지 판단 |
| **허위매물 감지** | 비정상적 가격 패턴 감지로 위험 매물 경고 |
| **협상 포인트 제공** | 데이터 기반 네고 전략 및 근거 제시 |

### 3.3 신뢰도 계산

```python
def calculate_confidence(similar_count, price_std, data_freshness):
    """
    신뢰도 산출 공식
    
    요소:
    1. 유사 매물 수 (40%)
    2. 가격 표준편차 (35%)
    3. 데이터 최신성 (25%)
    """
    
    # 유사 매물 수 점수 (0-100)
    count_score = min(similar_count / 50, 1.0) * 100
    
    # 가격 편차 점수 (편차 낮을수록 높은 점수)
    std_score = max(0, 100 - (price_std / 예측가 * 100))
    
    # 데이터 최신성 (최근 30일 데이터 비율)
    freshness_score = (recent_data_ratio) * 100
    
    # 가중 평균
    confidence = (count_score * 0.4 + 
                  std_score * 0.35 + 
                  freshness_score * 0.25)
    
    return round(confidence, 1)
```

### 4.3 세부 점수 계산

#### 금리 점수 (25점)

```python
def calculate_rate_score(current_rate, trend):
    """
    금리가 낮거나 하락 추세일 때 높은 점수
    
    기준:
    - 2.0% 이하: 25점 (매우 좋음)
    - 2.5% 이하: 20점 (좋음)
    - 3.0% 이하: 15점 (보통)
    - 3.5% 이하: 10점 (나쁨)
    - 그 이상: 5점 (매우 나쁨)
    
    추세 보정:
    - 하락 추세: +3점
    - 상승 추세: -3점
    """
    base_score = get_base_score(current_rate)
    trend_adjustment = get_trend_adjustment(trend)
    
    return min(25, max(0, base_score + trend_adjustment))
```

#### 유가 점수 (25점)

```python
def calculate_oil_score(current_price, trend, fuel_type):
    """
    유가가 낮을수록 내연기관 차량 운용비 절감
    전기차/하이브리드는 유가 영향 적음
    
    기준 (WTI, USD/배럴):
    - $60 이하: 25점
    - $70 이하: 20점
    - $80 이하: 15점
    - $90 이하: 10점
    - 그 이상: 5점
    
    연료 타입 가중치:
    - 전기: 유가 영향 0%
```
